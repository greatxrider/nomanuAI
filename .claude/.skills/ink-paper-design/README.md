# Ink & Paper Design System

A refined editorial design system for the **Personal Action Blueprint** style. Features Newsreader serif typography, subtle off-white paper backgrounds, ink-toned text colors, and elegant gold accents.

## Overview

This skill provides comprehensive styling guidelines for creating websites and UI components with a sophisticated, editorial aesthetic.

### Design Characteristics

- **Editorial Typography**: Newsreader serif for headlines, Inter sans-serif for body text
- **Subtle Backgrounds**: Off-white `#FAFAF8` for section depth
- **Minimal Borders**: Thin `#E5E3DE` lines for clean separation
- **Gold Accents**: `#C5A572` for highlights and active states
- **Monospace Times**: Tabular time displays for schedules
- **Hover Elevations**: Cards lift with subtle shadows on interaction
- **Responsive Typography**: Fluid type scales using `clamp()`

## Typography

- **Headlines**: `Newsreader` (serif) - weights 400, 500, 600
- **Body/UI**: `Inter` (sans-serif) - weights 400, 500, 600
- **Japanese**: `Noto Serif JP` - weight 500
- **Times/Data**: `ui-monospace`
